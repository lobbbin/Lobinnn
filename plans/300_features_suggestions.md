11. **New Game Plus** - Start new game with accumulated legacy points but increased difficulty
12. **Ironman Mode** - No save/load, one life gameplay with global leaderboards
13. **Tutorial Mode** - Interactive tutorial teaching core mechanics step by step
14. **Quick Start Options** - Pre-configured country presets for quick gameplay
15. **Victory Conditions** - Multiple win conditions: economic dominance, military superpower, diplomatic influence, technological supremacy
